这是一个用于《更多女仆交互（More Maid Interaction）》的知识技能。

核心目标：把玩家行为（送礼、摸头、投喂、膝枕等事件）映射为：
- 女仆台词（沉浸优先、中文、短句、符合当前关系）
- 情绪/关系数值变化（轻量、可控、范围受限）
- 记忆标签（短字符串，用于后续剧情分支）

当你需要生成“事件反应”时，输出应满足开发者消息里约定的 JSON 结构。
